## NVDA Portfolio Performance & Risk Analysis

This project analyzes the return, risk, and trend behavior of NVIDIA (NVDA) using SQL-based time series analytics.

### Key Analytics
- Daily and cumulative returns
- Volatility (daily & annualized)
- Sharpe ratio
- Maximum drawdown
- Trend detection using 7-day and 20-day moving averages

### Tools
- SQL

### Key Insights
- NVDA exhibits high volatility (~50% annualized) with a maximum drawdown of ~37%.
- Moving-average trend analysis identifies momentum shifts useful for timing-based strategies.
- Results highlight the tradeoff between upside potential and downside risk for growth equities.

This project demonstrates portfolio analytics and risk management skills relevant to portfolio management roles.
